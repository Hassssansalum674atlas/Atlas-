# Atlas

Atlas is an AI-assisted smart glasses platform designed for training and real-world task guidance.

## Vision
To augment human capability through contextual AI assistance in training, safety, and field work.

## Modes
- Trainer Mode: step-by-step guidance, correction, and learning
- Working Mode: real-time task assistance and safety awareness

## Status
Concept and early architecture definition.

# Atlas – Roadmap

This document outlines the incremental development and deployment plan for Atlas.

## Phase 1 – Foundation
- Build core edge AI infrastructure
- Implement two initial modes: Trainer Mode and Working Mode
- Focus on one flagship module (Pocket OSHA)
- Validate operation on phones and existing smart glasses
- Establish GitHub documentation and legal footprint

## Phase 2 – Expansion
- Integrate additional modules (Microgrid Stabilizer, Portable EHR)
- Introduce partial edge AI for offline reasoning
- Add HUD overlays and improved latency handling
- Collect user feedback for iterative improvement

## Phase 3 – Scaling
- Deploy to multiple regions with localized data
- Expand to enterprise clients and partner programs
- Refine AI engine (Forge) and SDK capabilities
- Explore optional hardware integration (custom smart glasses)

## Build Principles
- Incremental capability growth
- Safety and privacy first
- Modular and extensible design
- Clear separation between Atlas platform and CorelTex company IP

*Note:* This roadmap represents **planned direction**, not guaranteed delivery dates.
